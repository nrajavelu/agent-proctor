🎯 Universal AI Proctoring SDK

Standalone iframe-based proctoring for ANY web application

✨ Key Features

🔌 Zero Integration
Just 2 lines of JavaScript. Works with React, Vue, vanilla JS, or any framework.
🎪 Self-Contained
Independent iframe with own WebSocket connection, session management, and UI.
💾 Persistent Sessions
Survives page reloads, navigation changes, and host app crashes.
🚨 Real-time Alerts
Instant violation notifications with severity-based styling.

🚀 Quick Integration

Add AI proctoring to any application in under 30 seconds:

<script src="https://proctor.yourdomain.com/sdk.js"></script> <script> // Initialize proctoring ProctorSDK.init({ candidateId: 'student001@cs.university.edu', examId: 'financial-literacy-demo', organizationId: 'cs-dept-uuid', sessionManager: 'ws://localhost:8080', enableSimulation: true // For testing }); </script>
